private-state

DApp CLI

Browser-based operator for private-state user and tester entrypoints. Serve this folder over HTTP, select a target network, load a function template from `calldata.json`, then generate calldata, call, or send a transaction through MetaMask.

Notes

  • The CLI reads function templates from `cli/functions/<function-name>/calldata.json`.
  • Duplicate low-signal getters such as the repeated `controller()` views are intentionally omitted to avoid folder name collisions.
  • MetaMask is required for live `eth_call` and `eth_sendTransaction` execution. Calldata generation works without a wallet.
