# Script Artifacts

This directory stores long-lived generated artifacts that are intentionally kept under version control or are meant to be consumed by other repository code, developers, or users.

## Contents

### `contracts/`

Network-specific contract manifests generated by deployment and upgrade scripts:

- `contracts-<network>.json`

These files contain deployed contract addresses together with their ABIs and are intended for downstream tooling and human inspection.

## Convention

- Long-lived generated artifacts belong under `script/artifacts/`.
- Temporary execution output belongs under an `output/` directory and should remain ignored by Git.
