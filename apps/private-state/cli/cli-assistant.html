Handoff Notes

Commands are built against node apps/private-state/cli/private-state-bridge-cli.mjs.
On anvil, the assistant omits --alchemy-api-key because the current CLI does not require it there.
For JSON-array arguments such as --amounts, --note-ids, and --recipients, keep valid JSON in the stored field.
True one-click shell execution would need a native bridge, browser extension, terminal protocol, or host application. This demo deliberately stops at safe command handoff.
