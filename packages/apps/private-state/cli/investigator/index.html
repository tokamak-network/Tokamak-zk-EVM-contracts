Private-State Evidence Investigator

Filter a local raw evidence ZIP into a narrower user-consent disclosure package.

Raw evidence bundles are not submission packages. This tool runs entirely in the browser. It reads a full-note evidence bundle, lets the user select a limited disclosure scope, and writes a new ZIP containing only selected records and directly referenced transaction evidence. It does not request or include viewing keys, spending keys, wallet secrets, or account private keys.
